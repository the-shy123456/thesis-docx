# Thesis Final Audit Checklist (Sample)

Use this as a pre-delivery checklist before saying the thesis formatting is
complete.

## Requirement Boundary

- School formatting guide loaded and understood
- User-specific style rules separated from school rules
- Unspecified formatting intentionally preserved

## Structure Audit

- Cover page
- Abstract / keywords
- English abstract / keywords
- TOC title and TOC levels
- Heading hierarchy
- Figure captions
- Table captions
- References
- Acknowledgements
- Section/page-number scheme

## Hidden OOXML Audit

- Style IDs verified for heading levels
- `firstLine` checked where indentation looks suspicious
- `firstLineChars` checked where indentation looks suspicious
- numbering indentation checked
- `titlePg` and first-page section settings checked
- REF field display text checked

## Word PDF Review

- Exported from Word to PDF
- Every page reviewed
- Dense figure/table pages reviewed carefully
- First page of each chapter reviewed
- Reference pages reviewed
- Final acknowledgement page reviewed

## Delivery Gate

- No unresolved format regressions remain
- If any page was not reviewed, that limitation is stated explicitly
